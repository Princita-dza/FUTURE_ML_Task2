**# Support Ticket Classification**



**## Project Overview**

**This project classifies customer support tickets into different categories and assigns priority levels using Machine Learning and Natural Language Processing (NLP).**



**## Features**

**- Text preprocessing**

**- Ticket category classification**

**- Priority prediction (High, Medium, Low)**

**- Model evaluation**



**## Technologies Used**

**- Python**

**- Pandas**

**- NumPy**

**- NLTK**

**- Scikit-learn**

**- Matplotlib**



**## Dataset**

**Customer support ticket dataset from Kaggle.**



**## Output**

**The system predicts the ticket category and priority level based on the ticket description.**
